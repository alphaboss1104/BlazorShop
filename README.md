## :eyes: Try it out

Make sure you have the [.NET Core SDK](https://www.microsoft.com/net/download/macos) installed on your machine. Then do:

```
git clone https://github.com/kalintsenkov/BlazorShop.git
cd BlazorShop
dotnet run
```
